# MOTIVATION_EXPERIMENTS 按日期归档规范

本目录是实验日志仓库的唯一日期归档根目录。日期目录使用实验首次创建或启动日期，格式固定为 YYYY-MM-DD。

## Canonical 路径

服务器 canonical 路径：/raid/home/hming/FusionRAG-pca-analysis/MOTIVATION_EXPERIMENTS/by_date

不要把 /home/hming/... 当成另一份目录；它与 /raid/home/hming/... 是同一挂载上的路径别名。实验文档中统一记录 /raid 路径。

## 目录规则

- 一个日期只允许一个目录，不复制同名日期目录。
- 新实验放入创建日期目录，不按模型、GPU 或 worker 另建日期目录。
- 每个日期目录必须有 README.md，列出当天实验子目录。
- 每个实验目录必须先有 plan.md，再运行；完成后维护 README.md 和 EXPERIMENT_LOG.md。
- 结果 CSV/图表/小型 JSON 可放实验日志仓库；大型 artifact 放到 experiment-artifacts，并在 ARTIFACT_LOCATIONS 文档登记。
- cache 不放本目录，统一按 MODEL/DATASET 复用。
- 软链接只能指向本目录内的 canonical 目标，并使用相对路径；禁止写入 /home 或 /raid 的绝对实验路径。
